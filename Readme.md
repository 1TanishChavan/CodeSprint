# Setup:

### Setup Backend:

```
cd backend
Run npm i
```

### Setup Database:

- postgres database

* Create .env file in bankend folder like .env.sample
* Run this command

```
cd backend
db:setup
```

### Setup Frontend:

```
cd frontend
Run npm i
```

# Run Application

terminal 1

```
cd frontend
npm run start
```

terminal 2

```
cd backend
npm run dev
```
